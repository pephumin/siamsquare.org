Option #1 (click to see the set up example)

  • Data collection using an online platform
  • Questions are all open-ended including 3 key questions (start/stop/continue) across 3 selected touch points
  • Report will be generated automatically using the same format for all managers
  • Customised access permission to the system to only allow each evaluator and/or each manager to see only the contents that he/she is supposed to see

Option #2 (click to see the set up example)

  • Data collection using an online platform
  • Questions are the same as option #1, plus one additional attribute rating within each section (start/stop/continue) in order to understand further on the depth of opinions provided
  • Report will be generated automatically using the same format for all managers with quantitative score shown within each section comparing with the total average or department average (TBC)
  • Customised access permission to the system to only allow each evaluator and/or each manager to see only the contents that he/she is supposed to see
